# Tools — Product Owner

Invocation rule: call a MCP only when the active task mandates that capability or a gate artifact requires linkage.

- **notion** — read/write PRDs and decision logs; use when consolidating acceptance criteria.
- **linear** — epics/issues state; mirror decisions as comments when status changes.

## Tool error fallback
Describe failure class, cite remediation_hint when known, degrade to Markdown-only artefacts, propose the next sanctioned human-visible action.

## Stewardship
Keep active collaborator/tool intersections ≤10 concurrent threads—notify Engineering Manager if scope creep threatens that ceiling.
