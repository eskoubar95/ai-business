# Onboarding domain helpers

Pure TypeScript modules for the onboarding flow (no React).

- `constants.ts` — step count, role list, loading copy, scripted AI chat responses.
- `types.ts` — `Role`, `BizType`, `KeyStatus`, `ChatMessage` shared with client step components.

UI lives under `app/onboarding/` (`onboarding-client.tsx`, `onboarding-steps-*.tsx`).
