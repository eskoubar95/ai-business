## Important Findings

- Use **`npm install --legacy-peer-deps`** (or `.npmrc` with `legacy-peer-deps=true`) for `@neondatabase/auth` with Next 15 until peer range aligns.
- **Session probe:** validating “401 vs unauthenticated” needs a reachable **Neon Auth** base URL (`NEON_AUTH_BASE_URL`). With an invalid/hostile URL the handler returns gateway-style errors instead of pure 401.
